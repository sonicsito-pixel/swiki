# Raw Sources

이 폴더는 원본 자료 보관소다. LLM은 이 폴더의 파일을 수정하지 않는다.

## Recommended Use

- `sources/`: 읽고 분석할 원문 자료를 넣는다.
- `assets/`: 이미지, PDF, 첨부파일, 도표 등을 넣는다.
