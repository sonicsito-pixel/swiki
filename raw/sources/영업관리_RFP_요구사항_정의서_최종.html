1. 매장 기준정보 및 이력 관리

요구사항 ID REQ-MDM-001
요구사항명 위치 기반 고유 매장코드 및 SCD 이력 관리 엔진
기능 상세 설명 - 사업자번호 변경과 무관하게 유지되는 물리적 거점 중심의 '영구 매장코드'를 PK로 설정
- SCD(Slowly Changing Dimension) Type 2 적용: 점주명, 사업자정보, 계약조건, 매장속성(면적 등)의 변경 이력을 타임스탬프 기반으로 누적 관리
- 과거 시점 조회 시 해당 시점의 매핑 속성을 자동 호출하여 시계열 데이터 정합성 보장
비즈니스 목적 점주 변경 시 데이터 단절 해소 및 매장별 생애주기 매출/성과 분석 자동화
요구사항 ID REQ-MDM-002
요구사항명 부서별 입력 탭(Tab) 기반 데이터 거버넌스
기능 상세 설명 - 영업(계약), 회계(금융/세무), 물류(배송지/창고) 등 부서별 전용 입력 영역 분리
- 필수 항목 미입력 시 다음 단계 진행을 차단하는 Workflow 승인 프로세스 구현
- 부서별 업무 R&R에 따른 필드 단위 권한 제어(Read/Write) 적용
비즈니스 목적 데이터 입력 누락 방지 및 부서 간 정보 간섭에 의한 오입력 원천 차단

2. 매출 확정 및 자동 정산 엔진

요구사항 ID REQ-SLS-001
요구사항명 Integration Hub 기반 매출/VAN 데이터 수집 엔진
기능 상세 설명 - 주요 유통망(롯데/현대/신세계 등) EDI 데이터 자동 스케줄링 수집
- 카드사/VAN사 입금 내역 API 실시간 연동 인터페이스 구축
- 외부 데이터 양식을 내부 표준 양식으로 자동 변환(ETL) 처리
비즈니스 목적 수기 엑셀 다운로드 및 재가공 공정 제거(정산 리드타임 80% 단축)
요구사항 ID REQ-SLS-002
요구사항명 3-Way Auto-Matching 및 예외 관리 대시보드
기능 상세 설명 - [ERP 매출] - [유통망 정산액] - [실제 입금액] 자동 3자 대조 엔진 구현
- 1원 단위 정합성 확인 건은 시스템 자동 확정(Zero-Touch Approval)
- 미매칭 건 발생 시 사유(취소 누락, 할인율 불일치 등) 자동 추천 및 조정 전표 생성 기능
비즈니스 목적 전수 조사 업무를 예외 관리 체계로 전환하여 업무 효율성 극대화

3. 실시간 재고 가시성 및 수불 관리

요구사항 ID REQ-INV-001
요구사항명 Hub-Spoke 실시간 논리적 수불 엔진
기능 상세 설명 - 거점(Hub)-종속(Spoke) 매장 간의 물리적/논리적 계층 관계 정의 기능
- 종속 매장 판매 시 거점 재고 자동 차감 및 실시간 매출 배분 로직 구현
- 가상 창고(X/OA) 없이 직접적인 점간 이동(RT) 트랜잭션 관리
비즈니스 목적 중간관리 매장 재고 투명성 확보 및 실시간 가용 재고 정확도 향상
요구사항 ID REQ-INV-002
요구사항명 현장 완결형 자가소모 직접 처리 프로세스
기능 상세 설명 - 매장 단말기(POS/App)에서 직접 자가소모(비품, 증정 등) 사유 입력 및 재고 차감
- 물류센터 반품 미경유 프로세스 설계
- 등록 시 해당 비용 계정(판촉비 등)으로 회계 전표 자동 연동 인터페이스
비즈니스 목적 불필요한 물류 비용(택배비 등) 절감 및 현장 비품 관리 신속성 확보

4. 액티브 리스크 및 채권 모니터링

요구사항 ID REQ-ACC-001
요구사항명 CMS 연동 실시간 채권 상계 및 자동 판매 제어
기능 상세 설명 - CMS 입금 데이터 수신 즉시 ERP 채권 잔액 자동 차감(Real-time Offset)
- 채권 잔액이 임계치(Threshold) 초과 시 시스템 로그인 또는 판매 기능을 즉시 차단하는 Auto-Blocking 로직
- 리스크 수준별 3단계 제어: 1단계(경고) → 2단계(판매제한) → 3단계(접속차단)
비즈니스 목적 데이터 가시성을 넘어선 실시간 사고 예방 체계 구축

5. 증빙 자동화 및 연동 관리

요구사항 ID REQ-DOC-001
요구사항명 Trustbill API 연동 및 정산 공제 통합 인터페이스
기능 상세 설명 - 정산 확정 시 외부 시스템(Trustbill)으로 데이터 자동 전송 및 전자세금계산서 역발행
- 부자재 구매(소모품), AS 수선비 등 타 부서 비용 발생 데이터의 자동 정산 공제 반영
- ERP 전표 화면 내 증빙 이미지(PDF) 및 국세청 전송 상태 통합 조회 뷰어
비즈니스 목적 부서 간 중복 업무 제거 및 증빙 누락에 따른 세무 리스크 원천 차단
