1. F-ONE 시스템 현황
FONE 시스템 가용 화면 ( 시스템 메뉴 등록 기준 ) : 총 383 개
조회화면 234개, 등록 화면 149개, 주요 데이터 등록화면 56개
미 사용 프로그램 63개 ( 현업 확인을 통해 화면 CLOSE 결정 필요 )
▶ 주요 데이터 등록 화면 분포
- 영업 관리 17개, F-POS 14개, 정보관리 8개, 물류관리 6개, 영업관리 6개, 기타 …
▶ 주요 데이터 항목
- 제품 정보 구성을 위한 기준정보 ( 브랜드, 칼라, 사이즈 정보 일체 )
- 물류 재고 수불 기본 정보
- 매장 재고 수불 기본 정보
- 매장 매출 및 정산 관련 데이터

2. 주요 정보 (데이터) 등록 도식화
□ 기준정보 및 제품코드 등록
- 브랜드, 기획 년도/시즌, 품종, 성별 정보 속성을 통해 제품 정보 생성
-> 작업 지시서 등록 시 제품 정보/스타일정보 생성
□ 물류 재고 수불 / WMS
- FONE 등을 통해 접수된 물류 이동 요청 정보를
WMS 예정정보로 등록하고 처리 후 실적 정보를 받아 FONE 시스템에 인터페이스 하여 물류 수불 정보 등록
-> 생산처 입고, 매장반품입고, 출고처리, 자가소모
□ 매출(판매) 등록 및 매장 재고 수불
- 매출 등록을 통한 판매 데이터를 POS 시스템으로부터 인터페이스 받아 매장 수불(판매) 정보  및 판매 영수증 정보 등록
- 매장 입고, 물류 반품, 점간 재고 이동 처리 등 매장 재고 관리에 대한 업무 등록을 진행하며 매장 수불 정보 및 거래명세서 정보 등록
□ 영업 관리 / 마감
- 매장 매출 정보, 매장 재고 수불, 입금정보, 공제 항목 정보, 중간 관리자 정보 및 수수료 정보, 매장 계약 정보를 집계하고, 취합/정리 하여 마감 작업 진행
-> 회계 시스템 ( AIS ) 전송 처리

3. 시스템 업무 흐름
FONE 의 각 시스템은 현업 업무에 맞춰진 카테고리로 구성 되어 있으며 SONE, WMS, POS, AIS 등 여러 기능적 업무 유관 시스템과 연결된 구조
* 기준 정보 체계 개선
- 브랜드, 품종 정보 등 기준정보 체계에 대한 유연성 강화
* 수불/재고 관리 강화
수불 추적성 확보를 통한 재고 정합성 제고
가격/프로모션 강화
가격 등록 이력관리
프로모션 정책 활용도 제고를 위한 시스템
정산 / 마감 편의성
정산 프로세스 자동화
수기 업무 시스템 화 및 데이터 정합성강화
