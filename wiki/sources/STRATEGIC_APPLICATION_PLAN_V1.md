## 1. 개요
2026년 5월 13일 진행된 외부 컨설턴트(TGCS 박병원 상무) 자문 내용을 바탕으로, 데상트코리아, K2, 신세계인터내셔날 등 동종업계 선진 사례를 신성통상 차세대 시스템(FA-ONE)에 최적화하여 적용하기 위한 전략적 로드맵을 제안함.

---

## 2. 핵심 영역별 적용 전략

### 2.1 기준정보(Master Data) 거버넌스 구축
- **[벤치마킹]** 전사 영향도 분석 기반의 MDM(Master Data Management) 중심 통제.
- **[자사 적용]**
    - **코드 체계 개편:** 브랜드 코드 확장(영숫자 조합) 시 신/구 코드 병행 운영 및 자연 소멸 전략 채택.
    - **거버넌스 정립:** '표준정보(IT)'와 '분석속성(MD)'의 관리 주체 명확화 및 데이터 입력 단계에서의 무결성 체크 로직 강화.

### 2.2 인프라 및 DB 현대화
- **[벤치마킹]** Oracle 유지 + OCI(Cloud) 전환을 통한 비용 절감 및 기술 안정성 확보 (데상트 사례).
- **[자사 적용]**
    - **Cloud Native 전환:** 기존 Oracle 자산(Procedure 등)을 보존하면서 OCI로 이전하여 클라우드 유연성 확보.
    - **파일 분리 저장:** AI Vector Search 대응을 위해 대용량 이미지/도면 파일은 Object Storage로 분리, DB 부하 최소화.

### 2.3 API 중심의 시스템 인터페이스 표준화
- **[벤치마킹]** EAI를 대체하는 RESTful API 기반 API Gateway 구조 채택.
- **[자사 적용]**
    - **통합 인터페이스 창구:** 외부 유통망 및 이커머스 연동을 API Gateway로 단일화하여 정합성 유지 및 유지보수 비용 절감.
    - **Headless ERP 지향:** 모든 비즈니스 로직을 API화하여 향후 모바일 및 신규 채널 확장에 즉각 대응 가능하도록 설계.

### 2.4 상품/영업 실무 기능 고도화
- **[벤치마킹]** 세트 단위 운영 로직(SI) 및 모바일 기반 현장 실무 플랫폼(K2).
- **[자사 적용]**
    - **세트 관리 내재화:** 기획-영업-정산 전 과정에 '세트(Set) 상품' 처리 로직을 기본 사양으로 정의.
    - **모바일 영업 플랫폼:** 매장 현장에서 재고 조정, RT(이동), 실시간 정산 확인이 가능한 모바일 앱 기능을 RFP 핵심 요구사항으로 반영.

## 3. 실행 로드맵 (Roadmap)

1.  **Phase 1 (ERP 재구축):** 데이터 구조 표준화 및 API 중심 연계 체계 수립.
2.  **Phase 2 (고도화):** AI 분석 엔진 결합 및 옴니채널 강화.
3.  **Phase 3 (DT 완성):** 모바일 기반 전사 실무 플랫폼 확산 및 AI 기반 의사결정 체계 정착.
